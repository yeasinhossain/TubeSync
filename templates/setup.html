How to get your API Credentials

TubeSync is 100% free because it uses your own Google Cloud quotas. Follow these 5 minutes steps to set it up.

1 Create a Google Cloud Project

Go to the Google Cloud Console. Click the project dropdown at the top and select New Project. Name it "TubeSync App" and create it.

2 Enable the YouTube API

Using the left sidebar, navigate to APIs & Services > Library. Search for "YouTube Data API v3", click on it, and click the blue Enable button.

[Screenshot: The YouTube Data API v3 Enable Button]

3 Configure the Consent Screen (Important!)

Go to APIs & Services > OAuth consent screen. Select External and click Create. Fill in the required app names and your email.

Crucial Step: On the "Test users" step, click + ADD USERS and add the exact Gmail address of the YouTube channel you want to edit. If you skip this, Google will block your login!

[Screenshot: The "+ ADD USERS" button on the Consent Screen]

4 Create the Credentials

Go to APIs & Services > Credentials. Click + Create Credentials at the top and select OAuth client ID. Choose Web application as the Application type.

[Screenshot: Selecting "Web application" from the dropdown]

6 Copy your Keys

A modal will appear containing your Client ID and Client Secret. Copy both of those values and paste them into the form on the left side of this page!
